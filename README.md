If `Atom`, `BitString`, `Integer` and `Float` values wanna be a boolean value, they can using `to_boolean/1` function.

## Installation

The package can be installed by adding `wannabe_bool` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:wannabe_bool, "~> 0.1"}]
end
```
